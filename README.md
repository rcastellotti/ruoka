# ruoka

ruoka is a platform to make eating better cool again!

## features

- a shopping cart
  - integration with local supermarkets
- a recipes section
  - suggestions based
    - on what you have in your fridge
    - what you alread ate for the current week
- a fridge/pantry section
- a store section
  - buy a meal plan
  - buy single recipes
- possibility to buy
- easily track
